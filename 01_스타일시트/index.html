스타일 시트의 종류

인라인 스타일시트

인라인 스타일시트 : 태그 내부에서 style이라는 속성을 선언하여 그 내부에서 스타일 선언명과 선언값으 각 태그마다 적용하고자 할때 사용

내부 스타일 시트

내부 스타일 시트 : HTML문서 내부에 style 태그를 선언하여 해당하는 선택자에 스타일을 적용하고자 할 때 사용

외부 스타일 시트

외부 스타일 시트 : HTML 문서 외부에 별도의 스타일을 선언하는 문서를 구성하여 스타일을 적용하고자 할 때 적용

스타일 시트의 우선순위

인라인 스타일 시트 > 내부 스타일 시트 (내부 및 외부의 우선 순위는 마지막에 작성된 스타일의 영향을 받는다)
